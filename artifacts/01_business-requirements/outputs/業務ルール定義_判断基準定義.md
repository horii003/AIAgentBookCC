### 業務ルール一覧

| ルールID | ルール内容 | 適用タイミング | 例外条件 | 備考 |
|---|---|---|---|---|
| BRL-01 | ユーザーの入力内容から申請種別（交通費精算申請・経費精算申請）を判断する | 判断前 | なし | 申請種別が特定できない場合はBRL-02を適用 |
| BRL-02 | 入力情報だけでは申請種別を特定できない場合、「交通費精算申請」「経費精算申請」の2択をユーザーに提示して確認する | 判断前 | なし | ユーザーの選択に従って対応する申請フローへ誘導する |
| BRL-03 | 申請種別ごとに必要な申請書と申請先を提示する | 判断後・実行前 | なし | 社内申請ルールに基づく |
| BRL-04 | 申請書の必須項目が不足している場合、不足項目を特定して利用者に質問する | 実行前 | なし | 必須項目がすべて揃うまで繰り返す |
| BRL-05 | 収集した情報から申請書の必須項目を自動入力する | 実行前 | なし | 入力値は収集情報から生成する |
| BRL-06 | 自動作成した申請書を利用者に提示し、内容確認の承認を得る | 実行後 | なし | 利用者が承認するまで提出しない |
| BRL-07 | 申請内容を社内申請ルールと照合し、ミス・不備を検出する | 実行後 | なし | 不備がある場合は利用者に通知する |
| BRL-08 | 差し戻しリスクが高い項目を事前に警告する | 実行後 | なし | 要件上未定義（差し戻しリスク閾値の詳細） |
| BRL-09 | 申請書の自動提出は禁止する。提出操作は必ず利用者が行う | 実行後 | なし | AIによる自動提出は一切行わない |
| BRL-10 | 申請ルールの例外・裁量判断が必要なケースは人にエスカレーションする | 実行前・実行後 | なし | エスカレーション先は要件上未定義 |
| BRL-11 | 交通費精算申請：移動情報（移動日・出発地・目的地・交通手段・費用）を一区間ずつ収集する | 申請情報収集時 | なし | 対応交通手段は電車・バス・タクシー・飛行機の4種類 |
| BRL-12 | 交通費精算申請：運賃データから交通費を自動計算する | 申請情報収集時 | なし | 電車は経路テーブル検索、バス・タクシー・飛行機は固定運賃 |
| BRL-13 | 交通費精算申請：申請は経費発生日から3ヶ月以内であること | 実行前 | なし | 3ヶ月超の場合は申請不可として通知する |
| BRL-14 | 交通費精算申請：交通費が10,000円を超える場合は事前に上長の承認が必要 | 申請チェック時 | なし | 上長承認済みであることの確認を行う |
| BRL-15 | 交通費精算申請：駅名を正規化する（「渋谷駅」→「渋谷」） | 申請情報収集時 | なし | 余分な接尾語を除去して正規化する |
| BRL-16 | 経費精算申請：領収書画像から店舗名・金額・日付・品目を自動抽出する | 申請情報収集時 | なし | 自動抽出に失敗した場合は手動入力を促す |
| BRL-17 | 経費精算申請：品目から経費区分（事務用品費・宿泊費・資格精算費・その他経費）を自動判断する | 申請情報収集時 | なし | 判断できない場合はユーザーに選択を促す |
| BRL-18 | 経費精算申請：申請は経費発生日から3ヶ月以内であること | 実行前 | なし | 3ヶ月超の場合は申請不可として通知する |
| BRL-19 | 経費精算申請：経費が5,000円を超える場合は事前に上長の承認が必要 | 申請チェック時 | なし | 上長承認済みであることの確認を行う |
| BRL-20 | 交通費精算申請・経費精算申請ともに、すべての申請に業務目的の記載が必要 | 実行前 | なし | 業務目的が未記載の場合は入力を促す |

### 判断基準表

| 判断ID | 判断項目 | 入力情報 | 判断条件 | 判断結果 | 次アクション |
|---|---|---|---|---|---|
| JD-01 | 申請種別の特定可否 | ユーザーの申請内容（自由文） | 入力内容から交通費精算申請・経費精算申請のいずれかが特定できる場合 | OK | 特定した申請種別・申請書・申請先を提示する |
| JD-02 | 申請種別の特定可否 | ユーザーの申請内容（自由文） | 入力内容から申請種別が特定できない場合 | NG | 「交通費精算申請」「経費精算申請」の2択をユーザーに提示して確認する |
| JD-03 | 申請書必須項目の充足確認 | 収集済み申請情報・申請書の必須項目定義 | 必須項目がすべて揃っている場合 | OK | 申請書の自動作成を実行する |
| JD-04 | 申請書必須項目の充足確認 | 収集済み申請情報・申請書の必須項目定義 | 必須項目が1件以上不足している場合 | NG | 不足項目を特定して利用者に質問する |
| JD-05 | 申請ルール適合性 | 申請書内容・社内申請ルール | 申請内容がすべての申請ルールに適合する場合 | OK | 申請書を利用者に提示して確認を促す |
| JD-06 | 申請ルール適合性 | 申請書内容・社内申請ルール | 申請内容が1件以上の申請ルールに不適合の場合 | NG | 不備箇所を利用者に通知し修正を促す |
| JD-07 | 差し戻しリスク評価 | 申請書内容・差し戻しリスク判定基準 | 差し戻しリスクが高い項目がある場合 | NG | 警告メッセージを利用者に提示する |
| JD-08 | エスカレーション要否 | 申請内容・例外ケース判定基準 | 申請ルールの解釈に裁量判断が必要な場合 | NG | 人（申請担当者・上長等）にエスカレーションする |
| JD-09 | 交通費精算：3ヶ月以内確認 | 経費発生日・申請日 | 経費発生日から3ヶ月以内の場合 | OK | 申請を受け付ける |
| JD-10 | 交通費精算：3ヶ月以内確認 | 経費発生日・申請日 | 経費発生日から3ヶ月超の場合 | NG | 申請不可を通知する |
| JD-11 | 交通費精算：上長承認要否 | 交通費合計金額 | 交通費が10,000円以下の場合 | OK | 上長承認なしで申請可能 |
| JD-12 | 交通費精算：上長承認要否 | 交通費合計金額 | 交通費が10,000円超の場合 | 要承認 | 上長承認済みであることを確認する |
| JD-13 | 経費精算：3ヶ月以内確認 | 経費発生日・申請日 | 経費発生日から3ヶ月以内の場合 | OK | 申請を受け付ける |
| JD-14 | 経費精算：3ヶ月以内確認 | 経費発生日・申請日 | 経費発生日から3ヶ月超の場合 | NG | 申請不可を通知する |
| JD-15 | 経費精算：上長承認要否 | 経費金額 | 経費が5,000円以下の場合 | OK | 上長承認なしで申請可能 |
| JD-16 | 経費精算：上長承認要否 | 経費金額 | 経費が5,000円超の場合 | 要承認 | 上長承認済みであることを確認する |
