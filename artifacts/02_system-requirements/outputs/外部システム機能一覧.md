## 2. 外部連携システム分類（参考・任意）

- 基幹系（ERP／会計／人事）：なし（要件上未定義）
- 業務SaaS（申請／ワークフロー／CRM）：なし（申請先への提出は人が手動実施）
- 情報系（DWH／検索／ナレッジ）：なし（ナレッジはシステムプロンプト埋め込み）
- 通知系（メール／チャット）：なし（要件上未定義）
- 認証・認可（IdP）：なし（要件上未定義）
- 外部公共サービス／外部API：EXT-001（LLM推論サービス）

## 2. インタフェース詳細定義（要件定義レベル）

### 2.1 IF-ID：IF-001
### API名：LLM推論API

---

## 2.1.1 基本情報（👤）

| 項目 | 内容 |
|---|---|
| 外部システムID | EXT-001 |
| API提供元 | 要件上未定義（Claude API等のLLMプロバイダ） |
| 機能概要 | 申請種別判断・必須情報充足判断・申請書チェック・自然文生成等のLLM推論 |

| 想定利用エージェント | AG-001, AG-002, AG-003 |

---

## 2.1.2 I/F方式・非機能（👤）

| 項目 | 内容 |
|---|---|
| プロトコル | HTTPS |
| データ形式 | JSON |
| 認証方式 | API Key |
| レート制限 | 要件上未定義（LLMプロバイダのプラン依存） |
| SLA | 要件上未定義 |
| 冪等性 | 不要（LLM出力は確率的） |
