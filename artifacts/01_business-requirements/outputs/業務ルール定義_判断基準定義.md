### 業務ルール一覧

| ルールID | ルール内容 | 適用タイミング | 例外条件 | 備考 |
|---|---|---|---|---|
| BRL-01 | 申請ルールを参照して回答・判断する | 判断前 | なし | 申請ルールに定義されていない内容は推測・補完しない（BRL-14参照） |
| BRL-02 | 申請書作成に必要な不足情報を対話で確認する | 実行前 | なし | |
| BRL-03 | 申請内容の妥当性チェックを申請書作成後に実施する | 実行後 | なし | |
| BRL-04 | 申請種別（交通費精算申請・経費精算申請）が判断できない場合は選択肢を提示する | 判断前 | なし | ユーザーに「交通費精算申請」「経費精算申請」の選択肢を提示し、ユーザーの選択に従って対応するフローへ誘導する |
| BRL-07 | 必要情報が全て揃ってから申請書を生成する | 実行前 | なし | |
| BRL-08 | 領収書画像がある場合はOCRで情報を自動抽出する | 実行前 | 画像が不鮮明または非対応形式の場合は再アップロードを促す | |
| BRL-09 | 品目から経費区分を自動判断する | 実行前 | 判断困難な場合は社員に確認する | 経費精算申請のみ適用。区分：事務用品費・宿泊費・資格精算費・その他経費 |
| BRL-10 | 交通費精算申請の移動情報は一区間ずつ収集する | 実行前 | なし | |
| BRL-11 | 申請期限超過を検出した場合は警告する | 実行前 | なし | 経費発生日から3ヶ月以内 |
| BRL-12 | 高額申請の場合は事前に上長の承認が必要な旨を案内する | 実行前 | なし | 交通費精算：10,000円超、経費精算：5,000円超 |
| BRL-13 | 業務目的の記載は全申請種別で必須とする | 実行前 | なし | |
| BRL-14 | 申請ルールに定義されていない内容は推測・補完しない | 判断前 | なし | |
| BRL-16 | 申請書作成後に申請内容チェック（BIZ-06）を自動実行する | 実行後 | なし | |
| BRL-17 | 運賃データから交通費を自動計算する | 実行前 | 運賃テーブルに該当する経路が存在しない場合は手動入力を促す | 電車：経路テーブル検索による自動計算、バス/タクシー/飛行機：固定運賃 |
| BRL-18 | 駅名を正規化する | 実行前 | なし | 「渋谷駅」→「渋谷」のように、駅名から「駅」の接尾辞を除去して正規化する |

### 判断基準表

| 判断ID | 判断項目 | 入力情報 | 判断条件 | 判断結果 | 次アクション |
|---|---|---|---|---|---|
| JD-01 | 申請種別の特定（単一） | 申請意図テキスト、社内申請ルール | 申請意図が1つの申請種別（交通費精算申請／経費精算申請）に対応する | OK（特定済） | 申請種別・申請先を提示する |
| JD-02 | 申請種別の特定（判断不能） | 申請意図テキスト、社内申請ルール | 申請意図から交通費精算申請・経費精算申請のどちらか判断できない | 判断不能 | 「交通費精算申請」「経費精算申請」の選択肢をユーザーに提示し、選択に従って対応するフローへ誘導する |
| JD-03 | 申請種別の特定（不明） | 申請意図テキスト、社内申請ルール | 申請意図が交通費精算申請・経費精算申請のいずれにも該当しない | NG | 管理部門へのエスカレーションを案内する |
| JD-05 | 不足情報の有無 | 申請書作成必要情報一覧、収集済み情報 | 未収集の必須項目が1つ以上存在する | NG | 不足情報を対話で確認する |
| JD-07 | 経費区分の判断 | 品目名称、社内申請ルール | 品目が申請ルールの経費区分（事務用品費・宿泊費・資格精算費・その他経費）に合致する | 区分特定 | 申請書に経費区分を反映する |
| JD-08 | 全必要情報の収集完了 | 申請書作成必要情報一覧、収集済み情報 | 全必須項目が収集済み | 収集完了 | 申請書を自動生成する |
| JD-09 | 申請期限の妥当性 | 経費発生日、申請日 | 申請日 - 経費発生日 > 3ヶ月 | NG | 申請期限超過の警告を提示する |
| JD-10 | 高額申請の事前承認要否（交通費精算） | 申請金額 | 申請金額 > 10,000円 | 事前承認必要 | 上長への事前承認が必要な旨を通知する |
| JD-11 | 高額申請の事前承認要否（経費精算） | 申請金額 | 申請金額 > 5,000円 | 事前承認必要 | 上長への事前承認が必要な旨を通知する |
| JD-12 | 業務目的の記載有無 | 申請書（下書き） | 業務目的欄が空欄 | NG | 業務目的の記載を促す |
| JD-13 | 申請内容の総合妥当性 | 申請書（下書き）、社内申請ルール | 全項目が申請ルールに適合している | OK/NG | NG：違反箇所・修正案を提示する |
| JD-15 | 交通手段の判断 | 移動情報テキスト | 交通手段が電車・バス・タクシー・飛行機の4種類のいずれかに該当する | 交通手段特定 | 該当する交通手段で移動情報を登録する |
| JD-16 | 運賃の自動計算可否 | 交通手段、出発地、目的地 | 電車の場合：経路テーブルに該当経路が存在する。バス/タクシー/飛行機の場合：固定運賃データが存在する | 自動計算可/手動入力必要 | 自動計算可の場合：運賃を自動算出して反映する。手動入力必要の場合：ユーザーに運賃入力を促す |
| JD-17 | 経費区分の判断 | 品目名称、社内申請ルール | 品目が事務用品費・宿泊費・資格精算費・その他経費のいずれかに分類できる | 区分特定/その他経費 | 区分特定の場合：対応する経費区分を申請書に反映する。判断困難の場合：社員に確認する |
