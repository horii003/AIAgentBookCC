## ターゲットユーザー
- 一般社員（申請ルール・申請方法を十分に把握していない社員）

## 要求事項

- RQ-01: 申請意図を入力すると、交通費精算申請書・経費精算申請書のいずれかを提示してほしい
- RQ-02: 申請ルールや申請方法が分からなくても正しく申請できるようにしたい
- RQ-03: 不足情報を対話で確認しながら申請書を自動作成してほしい
- RQ-04: 申請内容をチェックして申請ミスや差し戻しを減らしたい
- RQ-05: 申請ルール・申請方法を問い合わせて確認できるようにしたい

## 業務要件

| ID    | 業務要件 | 関連要求  | AI関与 | 優先度 |
| ----- | -------- | --------- | ------ | ------ |
| BR-01 | 申請意図から必要な申請種別（交通費精算申請・経費精算申請）・申請書・申請先を判断して社員に提示する | RQ-01, RQ-02 | 実行 | High |
| BR-02 | 不足情報を対話で確認しながら申請書を自動作成する | RQ-03 | 実行 | High |
| BR-03 | 申請内容を社内申請ルールと照合してチェックし、問題点と修正案を提示する | RQ-04 | 実行 | High |
| BR-04 | 申請ルール・申請方法を問い合わせて回答する | RQ-05 | 参照 | Medium |

## 業務種別ごとの業務ルール

### 申請種別判断の業務ルール

| ID | 業務ルール | 詳細 | 優先度 |
| -- | ---------- | ---- | ------ |
| APP-01 | 申請意図に基づき申請種別を判断する | 社内申請ルールを参照し、申請意図に対応する申請種別（交通費精算申請／経費精算申請）を特定する | High |
| APP-02 | 申請意図から申請種別が判断できない場合はユーザーに選択肢を提示する | 交通費精算申請・経費精算申請のどちらか判断できない場合は、ユーザーに「交通費精算申請」「経費精算申請」の選択肢を提示し、ユーザーの選択に従って対応する申請フローへ誘導する | High |
| APP-03 | 申請ルールに定義されていない申請種別は推測・補完しない | 判断根拠を社内申請ルールに限定し、独自推測による誤案内を防止する | High |

### 申請書作成の業務ルール

| ID | 業務ルール | 詳細 | 優先度 |
| -- | ---------- | ---- | ------ |
| FORM-01 | 必要情報が全て揃ってから申請書を生成する | 申請書作成に必要な項目がすべて収集済みの場合のみ申請書を自動作成する | High |
| FORM-02 | 不足情報は段階的に確認する | 社員の入力負担を軽減するため、1回の対話で確認する項目数を制限する | Medium |
| FORM-03 | 経費精算申請に領収書がある場合はOCRで情報を自動抽出する | 領収書画像から日付・金額・支払先を抽出し申請書に自動反映する | Medium |
| FORM-04 | 交通費精算申請の移動情報は一区間ずつ収集する | 移動区間を一つずつ確認することで漏れを防止する。対応交通手段は電車・バス・タクシー・飛行機の4種類。運賃データから交通費を自動計算する（電車は経路テーブル検索、バス/タクシー/飛行機は固定運賃）。電車区間では駅名を正規化する（例：「渋谷駅」→「渋谷」） | High |
| FORM-05 | 業務目的の記載は全申請種別で必須とする | 全申請書に業務目的を必ず記載させ、差し戻し理由の削減を図る | High |

### 申請内容チェックの業務ルール

| ID | 業務ルール | 詳細 | 優先度 |
| -- | ---------- | ---- | ------ |
| CHK-01 | 申請書作成後に自動で申請内容チェックを実行する | 申請書（下書き）生成後、社内申請ルールとの照合チェックを自動実行する | High |
| CHK-02 | 申請期限超過を検出した場合は警告を提示する | 経費発生日から3ヶ月を超えた申請は警告する | High |
| CHK-03 | 高額申請の場合は事前に上長の承認が必要である旨を案内する | 交通費精算：10,000円超、経費精算：5,000円超の申請は事前に上長の承認が必要な旨を通知する | High |
| CHK-04 | チェック結果はOK/NGと修正案を合わせて提示する | NGの場合は違反箇所と違反ルール、修正案を明示する | High |

## 申請対象

### 経費精算申請の対象経費区分

| 経費区分 | 対象品目例 |
|---|---|
| 事務用品費 | 書籍、文房具、オフィス用品など |
| 宿泊費 | ホテル、宿泊施設など |
| 資格精算費 | 資格試験、受験料、認定費用など |
| その他経費 | 上記以外の業務関連経費 |

### 交通費精算申請の対象交通手段

| 交通手段 | 運賃計算方法 |
|---|---|
| 電車 | 経路テーブル検索による自動計算 |
| バス | 固定運賃 |
| タクシー | 固定運賃 |
| 飛行機 | 固定運賃 |
