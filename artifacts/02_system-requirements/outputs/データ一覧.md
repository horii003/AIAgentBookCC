## 2. データ詳細定義（データID単位：任意）

### 2.1 データID：DATA-001（データ名：社内申請ルール）
- 種別：ナレッジ/ファイル
- 概要：申請種別・必要項目・申請先・申請期限・高額閾値・経費区分・対応交通手段等の社内申請に関するルール定義。全エージェントが判断・チェック・照会の根拠として参照する
- 形式（拡張子/スキーマ）：テキスト（Markdown/JSON等）
- 保管場所：ファイルストア（EXT-003）
- 更新頻度：要件上未定義
- 参照方法（API/検索/ダウンロード等）：ファイル参照（全文ロード）
- データ分類・機密区分：D-1（社内）/ 社内
- マスキング要件：不要
- 保持期間：要件上未定義
- 品質要件（欠損率/更新遅延/正確性など）：最新版が常に参照可能であること。申請種別・閾値等の定義が正確であること
- 監査要件：参照操作は標準ログ（LOG-STD）に記録する

#### スキーマ（要件定義レベル：代表項目のみ）
| No | 項目名 | 型（概略） | 必須 | 説明 | 例 |
|---:|---|---|---|---|---|
| 1 | 申請種別 | enum | ○ | 対象の申請種別 | 交通費精算申請、経費精算申請 |
| 2 | 申請先 | string | ○ | 申請を提出する部門 | 経理部 |
| 3 | 申請期限 | string | ○ | 経費発生日からの申請期限 | 90日以内 |
| 4 | 高額閾値（交通費精算） | number | ○ | 事前上長承認が必要な金額 | 10000 |
| 5 | 高額閾値（経費精算） | number | ○ | 事前上長承認が必要な金額 | 5000 |
| 6 | 経費区分定義 | array | ○ | 経費精算申請の対象経費区分一覧 | [事務用品費, 宿泊費, 資格精算費, その他経費] |
| 7 | 対応交通手段 | array | ○ | 交通費精算申請の対象交通手段一覧 | [電車, バス, タクシー, 飛行機] |

## 3. AIエージェント向け追加項目（ファイルベース参照系データ）

本システムはRAGを使用しない。ナレッジ（DATA-001, DATA-012）はファイルストア上のファイルを全文ロードして参照する。

### DATA-001（社内申請ルール）

| 項目 | 内容 |
|---|---|
| 参照対象範囲 | 申請種別・必要項目・申請ルール全般 |
| 参照方式 | ファイル全文ロード |
| ファイル管理 | 版・発効日をファイル名またはファイル内ヘッダーで管理（詳細は設計フェーズで定義） |
| 根拠提示 | ルール名・申請ルール章番号（Lv2根拠提示：参照元提示） |
| 更新検知 | 要件上未定義 |
