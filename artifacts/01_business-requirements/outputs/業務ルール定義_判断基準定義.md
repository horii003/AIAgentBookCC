### 業務ルール一覧

| ルールID | ルール内容 | 適用タイミング | 例外条件 | 備考 |
|---|---|---|---|---|
| BRL-01 | 社員が申請内容を入力した場合、社内申請ルールを参照して申請種別を判断する | 判断前 | なし | BR-01, BR-03対応 |
| BRL-02 | 判断した申請種別に対応する申請書と申請先を提示する | 判断後 | なし | BR-02対応 |
| BRL-03 | 1つの申請内容に複数の申請種別が該当する場合は、すべての申請種別を提示する | 判断後 | なし | JUDGE-02対応 |
| BRL-04 | 申請書の生成前に必須項目がすべて収集されていることを確認する | 実行前 | なし | BR-05, FORM-03対応 |
| BRL-05 | 必須項目が未収集の場合は申請書生成を行わず、不足情報を対話で確認する | 実行前 | なし | BR-04対応 |
| BRL-06 | 申請書を生成する場合は申請種別に対応したテンプレートを使用する | 実行時 | なし | FORM-01対応 |
| BRL-07 | 申請書の生成後に申請ルール適合チェックを実行する | 実行後 | なし | BR-06対応 |
| BRL-08 | チェックで不備が発見された場合は、具体的な修正点を提示する | 実行後 | なし | BR-07, CHECK-04対応 |
| BRL-09 | 申請内容から申請種別（交通費精算申請・経費精算申請）を判断できない場合は、ユーザーに選択肢を提示して確認する | 判断前 | なし | BR-08対応 |
| BRL-10 | 交通費精算は10,000円超、経費精算は5,000円超の場合に上長承認が必要な旨を提示する | 実行前 | なし | CHECK-03対応 |
| BRL-11 | 交通費精算申請：移動日・出発地・目的地・交通手段・費用を一区間ずつ収集する | 実行時 | なし | TRANS-01対応 |
| BRL-12 | 交通費精算申請：対応交通手段は電車・バス・タクシー・飛行機の4種類に限定する | 実行前 | なし | TRANS-02対応 |
| BRL-13 | 交通費精算申請：電車は経路テーブル検索、バス・タクシー・飛行機は固定運賃で交通費を自動計算する | 実行時 | なし | TRANS-03対応 |
| BRL-14 | 申請は経費発生日から90日以内であること。超過した場合は申請不可を案内する | 判断前 | なし | TRANS-04, EXPENSE-03対応 |
| BRL-15 | 入力された駅名を正規化する（例：「渋谷駅」→「渋谷」） | 実行前 | なし | TRANS-06対応 |
| BRL-16 | 経費精算申請：領収書画像から店舗名・金額・日付・品目を自動抽出する | 実行時 | なし | EXPENSE-01対応 |
| BRL-17 | 経費精算申請：品目から経費区分（事務用品費・宿泊費・資格精算費・その他経費）を自動判断する | 実行時 | なし | EXPENSE-02対応 |
| BRL-18 | すべての申請に業務目的の記載が必要 | 実行前 | なし | TRANS-07, EXPENSE-05対応 |

### 判断基準表

| 判断ID | 判断項目 | 入力情報 | 判断条件 | 判断結果 | 次アクション |
|---|---|---|---|---|---|
| JD-01 | 申請種別の特定可否 | 社員の申請内容テキスト、社内申請ルール | 申請内容が社内申請ルールに定義された申請種別に該当する | OK | 該当申請種別・申請書・申請先を提示する |
| JD-02 | 申請種別の判断不能 | 社員の申請内容テキスト | 申請内容から交通費精算申請・経費精算申請のいずれかを判断できない | NG | 「交通費精算申請」「経費精算申請」の選択肢を提示し、ユーザーの選択に従って対応フローへ誘導する |
| JD-03 | 複数申請種別の該当 | 社員の申請内容テキスト、社内申請ルール | 申請内容が2種別以上の申請種別に該当する | OK | すべての該当申請種別・申請書・申請先を提示する |
| JD-04 | 申請書作成に必要な情報の充足 | 収集済み申請情報、申請書テンプレート必須項目 | すべての必須項目が収集済み | OK | 申請書を生成する |
| JD-05 | 申請書作成に必要な情報の充足（不足あり） | 収集済み申請情報、申請書テンプレート必須項目 | 1件以上の必須項目が未収集 | NG | 不足項目を質問して情報収集を継続する |
| JD-06 | 申請内容のルール適合 | 生成した申請書内容、社内申請ルール | 申請内容が社内申請ルールのすべての条件を満たす | OK | 申請書を提示し提出を案内する |
| JD-07 | 申請内容のルール不適合 | 生成した申請書内容、社内申請ルール | 申請内容が社内申請ルールの1件以上の条件を満たさない | NG | 不備・修正点を具体的に提示する |
| JD-08 | 申請金額の上限適合 | 申請金額、承認権限・上限規定 | 申請金額が承認権限・上限規定の範囲内 | OK | 次のチェックへ進む |
| JD-09 | 申請金額の上限超過 | 申請金額 | 申請金額が上限を超過（交通費：10,000円超、経費：5,000円超） | NG | 超過している旨と上長への事前承認が必要なことを提示する |
| JD-10 | 申請期限の適合（交通費） | 申請日、経費発生日 | 経費発生日から90日以内 | OK | 申請を継続する |
| JD-11 | 申請期限の超過（交通費） | 申請日、経費発生日 | 経費発生日から90日超 | NG | 期限超過の旨を提示し申請不可を案内する |
| JD-12 | 申請期限の適合（経費） | 申請日、経費発生日 | 経費発生日から90日以内 | OK | 申請を継続する |
| JD-13 | 申請期限の超過（経費） | 申請日、経費発生日 | 経費発生日から90日超 | NG | 期限超過の旨を提示し申請不可を案内する |
| JD-14 | 領収書画像からの情報抽出成功 | 領収書画像 | 店舗名・金額・日付・品目がすべて抽出できた | OK | 抽出情報を確認提示する |
| JD-15 | 領収書画像からの情報抽出失敗 | 領収書画像 | 情報の一部またはすべてが抽出できなかった | NG | 手動入力を求める |
