### 業務ルール一覧

|---|---|---|---|---|

| BRL-02 | 申請に必要な必須項目がすべて揃っていない場合、申請書の作成を開始しない | 実行前 | なし | |

| BRL-06 | 交通費精算申請には移動日・出発地・目的地・交通手段・費用（自動計算）・業務目的が必須 | 実行前 | なし | |
| BRL-07 | 経費精算申請には経費区分（自動判断）・金額（自動抽出）・日付（自動抽出）・領収書画像・業務目的が必須 | 実行前 | なし | |
| BRL-11 | 交通費精算申請: 交通費が10,000円を超える場合は事前に上長の承認が必要 | 実行前 | なし | |

| BRL-14 | 交通費精算申請: 交通費の運賃自動計算。電車は経路テーブル検索、バス/タクシー/飛行機は固定運賃を使用する | 実行中 | なし | |
| BRL-15 | 交通費精算申請: 駅名正規化。入力された駅名から「駅」等の接尾語を除去して正規化する（例：「渋谷駅」→「渋谷」） | 実行中 | なし | |
| BRL-16 | 経費精算申請: 領収書画像から店舗名・金額・日付・品目を自動抽出する | 実行中 | なし | |

| BRL-18 | 経費精算申請: 経費が5,000円を超える場合は事前に上長の承認が必要 | 実行前 | なし | |

### 判断基準表

| 判断ID | 判断項目 | 入力情報 | 判断条件 | 判断結果 | 次アクション |
|---|---|---|---|---|---|
| JD-01 | 申請種別の特定 | 社員の申請意図（自然文） | 申請意図から交通費精算申請・経費精算申請のいずれかを特定できる | OK | 申請種別・申請書・申請先を提示する |
| JD-02 | 申請種別の特定（判断不可） | 社員の申請意図（自然文） | 申請意図から申請種別（交通費精算申請・経費精算申請）を一意に特定できない | NG | 「交通費精算申請」「経費精算申請」の2択選択肢を提示してユーザーに選択を求める。ユーザーの選択に従い対応する申請フローへ誘導する |
| JD-04 | 必須情報の充足確認 | 収集済み申請情報・申請種別 | 申請種別に応じた必須項目がすべて揃っている | OK | 申請書の自動作成を開始する |
| JD-05 | 必須情報の充足確認（不足あり） | 収集済み申請情報・申請種別 | 申請種別に応じた必須項目のうち1つ以上が未入力 | NG | 不足項目を特定して確認質問を提示する |
| JD-06 | 申請書チェック（必須項目） | 作成済み申請書 | 申請書のすべての必須項目が記入されている | OK | 整合性チェックに進む |
| JD-07 | 申請書チェック（必須項目不備） | 作成済み申請書 | 申請書に未記入の必須項目が1つ以上ある | NG | 不備項目を申請者にフィードバックし修正を促す |
| JD-08 | 申請書チェック（整合性） | 作成済み申請書 | 申請書内の情報（日付・金額・区間等）に矛盾がない | OK | 申請ルール適合性チェックに進む |
| JD-09 | 申請書チェック（整合性不備） | 作成済み申請書 | 申請書内の情報に矛盾が検出された | NG | 矛盾箇所を申請者にフィードバックし修正を促す |
| JD-11 | 申請期限チェック | 経費発生日・申請日 | 経費発生日から申請日までが90日以内 | OK: 続行 / NG: 期限超過を申請者に通知し申請を停止する |  |
| JD-12 | 上長承認要否判断（交通費精算申請） | 交通費合計金額 | 交通費合計が10,000円を超える | 要承認: 上長承認が必要である旨を案内する / 不要: 通常フローへ進む |  |
| JD-13 | 上長承認要否判断（経費精算申請） | 経費金額 | 経費金額が5,000円を超える | 要承認: 上長承認が必要である旨を案内する / 不要: 通常フローへ進む |  |
