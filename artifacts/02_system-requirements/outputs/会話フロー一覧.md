---

## 1. 基本情報（👤）

| 項目 | 内容 |
|---|---|
| 対象業務 | BIZ-01（申請種別判断・案内） |
| 対象ユースケースID | UC-01 |
| 業務フロー位置 | ST-BIZ01-01〜ST-BIZ01-02 |
| 主エージェント | AG-001 |
| 関与エージェント | なし |
| 人の役割 | 利用者（申請意図入力・判断不可時の選択・案内確認） |

| 完了条件 | 申請種別・申請書・申請先が社員に提示され、社員が確認できる状態になった |

## 2. 会話の前提条件・トリガー（👤）

| 項目 | 内容 |
|---|---|

| 中断・再開条件 | 社員が選択肢からの選択を拒否した場合は中断。申請ルール外のケースはエスカレーションへ遷移 |
